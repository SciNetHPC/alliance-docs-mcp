Page enfant de Service infonuagique de Calcul Canada

Les instances (machines virtuelles ou VMs) utilisées pour fournir des données génériques ou du contenu Web sont configurées selon les gabarits de type persistant (voir Gabarits d'instances) et sont démarrées depuis un volume.

*Pour créer un serveur de données partagées, consultez Serveurs FTP dans notre environnement infonuagique où il est aussi question de HTTP et SFTP.
*Pour créer un serveur web, consultez Création d'un serveur web dans notre environnement infonuagique.
